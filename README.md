# Hoja de Trabajo 7 - Algoritmos y Estructuras de Datos

**Desarrollado por:** Valeria Hernández 25086

---

Este proyecto usa un conjunto de palabras básicas en inglés con su traducción al español. Las palabras fueron elegidas porque son comunes en el uso diario y ayudan a formar frases simples, por lo que resultan útiles para la traducción de textos.

El vocabulario incluye palabras como:

- Sustantivos (personas, objetos, lugares)
- Verbos básicos
- Adjetivos
- Pronombres, artículos y conectores

Cada palabra en inglés tiene su equivalente en español, formando pares que el programa usa para realizar la traducción.
